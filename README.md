# MAKKAH TRAVELS — Travel Agency Management System

"Your Journey Begins With Trust."

## Project Overview
MAKKAH TRAVELS is a modern SaaS-style travel agency platform designed for travel agencies, Umrah/Hajj operators, and visa consultants. The system provides a seamless experience for customers to browse packages, submit bookings, and request visa services, while offering administrators a robust dashboard to manage operations.

## Features
- **Customer Portal**: Browse tour packages, view detailed itineraries, request custom travel, apply for visa services, and contact the agency.
- **Admin Dashboard**: Comprehensive management of tour packages, bookings, customer inquiries, visa requests, and destinations.

## Tech Stack
- **Frontend**: HTML5, CSS3, Bootstrap 5, JavaScript
- **Backend**: PHP
- **Database**: MySQL

## Installation
1. Clone the repository to your local server (e.g., XAMPP/WAMP `htdocs`).
2. Create a MySQL database named `makkah_travels`.
3. Import the schema provided in `database/schema.sql`.
4. Update the database credentials in `config/database.php`.
5. Run the application through your local server (e.g., `http://localhost/makkah-travels`).

## System Workflow
1. **User Discovery**: Customers visit the responsive frontend to explore premium Umrah and travel packages.
2. **Booking/Inquiry**: Users fill out sleek, modern forms for packages, custom tours, or visas.
3. **Data Processing**: PHP securely handles the form data and stores it in the MySQL database.
4. **Admin Management**: Administrators log into the dashboard to review customer requests, update package catalogs, and track engagement statistics.
